# CatchRobo_FW

## 仕様
- 使用マイコン
  - Teensy4.1

- 対応アクチュエータ
  - Robomaster M2006/M3508/GM6020
  - Dynamixel
  - (Robstride)
  - ブラシレスDC
  - サーボ
  
- 対応センサ
  - Robomaster内蔵センサ
  - Dynamixel内蔵センサ
  - AMTエンコーダ
  - 9軸センサ


## コミットメッセージ
| Prefix | 説明 |
| -- | -- |
| add | 新規（ファイル）機能追加 |
| remove | 削除（ファイル）|
| feat | 新機能の追加 | 
| fix | バグの修正 | 
| update | 機能修正（バグではない） |
| change | 仕様変更 | 
| disable | 無効化（コメントアウト等） |
| revert | 変更取り消し |
| test | テストコードの追加、修正 | 
| docs | ドキュメントの変更 | 
| clean | 整理（リファクタリング等） |
| style | コードのスタイルに関する変更 (フォーマット、セミコロンなど) | 
| build | ビルドシステムや外部依存関係に関する変更 |
